# Pyra (PyraOS)

Digital-twin O&M console for solar plants — Invertix Energy × AI Hackathon (EnerParc challenge).

Detect → explain → quantify € lost → recommend O&M action.

> ⚠️ The EnerParc dataset is restricted to the event and is **not** committed (see `.gitignore`). Load it locally.

## Stack (planned)
- Python offline pipeline (pandas / DuckDB / pvlib / LightGBM) → compact artifacts
- Next.js 16 + TS + Tailwind 4 OS-style desktop (cloned from the `aliquot` base)
- O&M Copilot via the Anthropic API

See the plan for details.
